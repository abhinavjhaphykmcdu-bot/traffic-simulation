# Traffic Flow Simulation using MCMC
Developed as part of the *Statistical Analysis of Physical Systems using Monte Carlo and MCMC* program (PaAC x QCG Open Projects 2025, IIT Roorkee).

## Overview
This project models traffic flow as an interacting particle system (cars on a highway) using Metropolis Monte Carlo methods.

## Features
- 1D traffic model with safe distance constraints
- Periodic boundary conditions (ring road)
- Emergence of traffic jams
- Visualization of system evolution

## Concepts Used
- Monte Carlo Methods
- Markov Chains
- Complex Systems

## Tools
- Python
- NumPy
- Matplotlib
